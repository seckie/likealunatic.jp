サイト名「Like@Lunatic」について

Lunaticとは「奇人・変人・大馬鹿者・正気ではない様」とかそんな意味です。すなわち「Like a Lunatic」で「まるで奇人みたいに」というような意味になります。当サイト名は、「奇人と言われてしまうくらい、物事を探求し続ける」という、私の決意表明のようなものです。
